# Premium Modern Church Website

A cinematic premium church website built with:

- Next.js 15+
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- GSAP

## Install

npm install

## Run Development Server

npm run dev

## Production Build

npm run build

## GitHub Deployment

git init
git add .
git commit -m "Initial premium church website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/church-website.git
git push -u origin main
